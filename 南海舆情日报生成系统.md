# 南海舆情日报生成系统

## Core Features

- CSV文件拖拽上传

- 数据清洗去重

- 数据库存储管理

- Dify工作流集成

- AI报告生成

- 报告下载导出

- 历史记录查询

## Tech Stack

{
  "Web": {
    "arch": "react",
    "component": "shadcn"
  },
  "Backend": "Python FastAPI + SQLAlchemy + PostgreSQL",
  "Integration": "Dify API (复用现有Python脚本逻辑)"
}

## Design

采用Material Design设计语言，深蓝色主色调配合浅蓝色辅助色，卡片式布局突出专业性。包含主控制台的文件上传区、处理状态面板、历史记录列表，以及报告详情页的概览卡片、数据图表和内容展示区域。
